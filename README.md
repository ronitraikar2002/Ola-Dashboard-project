## Project Overview
Ola Ride Insights is a data analytics project that analyzes ride-sharing data to uncover trends in bookings, revenue, cancellations, and ratings. The project focuses on SQL-based analysis and interactive Power BI dashboards to generate actionable business insights.

This project demonstrates strong fundamentals in **data cleaning, MIS reporting, SQL analysis, and business intelligence**.

---

## Domain
Ride-Sharing & Mobility Analytics

---

## Problem Statement
Ride-sharing platforms generate large volumes of data related to ride bookings, pricing, vehicle types, and customer feedback. Analyzing this data helps businesses optimize operations, reduce cancellations, and improve customer experience.  
This project analyzes Ola ride data to identify booking patterns, revenue drivers, cancellation reasons, and rating trends to support data-driven decisions.

---

## Objectives
- Analyze ride booking trends and peak demand hours  
- Understand revenue distribution and payment methods  
- Identify ride cancellation reasons  
- Evaluate customer and driver ratings  
- Build dashboards for MIS and management reporting  

---

## Tools & Technologies
- **SQL** â€“ Data querying and analysis  
- **Python (Pandas, NumPy)** â€“ Data cleaning and EDA  
- **Power BI** â€“ Interactive dashboards and KPIs  
- **Excel** â€“ Data validation and reporting  

---

## Dataset
- Single Ola ride booking dataset containing:
  - Ride status and timestamps  
  - Vehicle types  
  - Payment methods  
  - Booking value  
  - Customer and driver ratings  
  - Cancellation reasons  

---

## Project Workflow

### 1. Data Understanding & Exploration
- Reviewed dataset structure and key fields  
- Performed exploratory data analysis (EDA)  

### 2. Data Cleaning & Preprocessing
- Handled missing and inconsistent values  
- Standardized date, time, and numeric fields  
- Created derived metrics for reporting  

### 3. SQL Analysis
- Wrote SQL queries to analyze:
  - Successful vs cancelled rides  
  - Revenue and booking value  
  - Vehicle-wise performance  
  - Payment method usage  
  - Customer and driver ratings  

### 4. Power BI Dashboard
- Ride volume over time  
- Booking status breakdown  
- Revenue by payment method  
- Vehicle type performance  
- Cancellation reasons  
- Ratings distribution  

---

## Key Business Insights
- Identified peak booking periods for better driver planning  
- Highlighted major cancellation reasons affecting service quality  
- Analyzed revenue contribution by payment method  
- Evaluated vehicle types based on ride distance and ratings  

---

## Deliverables
- Cleaned dataset  
- SQL query files  
- Interactive Power BI dashboard  
- Project documentation  

---

## Outcome
